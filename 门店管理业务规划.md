# 美发预约记账系统 - 门店管理业务规划

## 1. 业务背景

当前系统存在的问题：
- 菜单（数据总览、预约管理、记账管理、服务管理、顾客管理、营业设置、营收统计）都是门店级别的功能
- 缺少完整的门店入驻审核流程
- 缺少店长申请入驻的入口

## 2. 目标业务流程

## 3. 角色权限设计

## 6. API 接口设计

### 6.1 店长申请入驻接口

```typescript
// POST /api/merchant/apply
// 小程序端调用
interface ApplyMerchantRequest {
  name: string           // 门店名称
  address: string        // 地址
  phone: string          // 联系电话
  business_hours_start: string  // 营业开始时间
  business_hours_end: string    // 营业结束时间
  description?: string   // 门店简介
  applicant_name: string // 申请人姓名
  applicant_phone: string // 申请人手机号
}

// 响应
{
  code: 0,
  message: '申请已提交，请等待审核',
  data: {
    application_id: string
    status: 'applying'
  }
}
```

## 8. 数据库初始化流程

### 8.1 门店创建时自动初始化

```typescript
async function initializeMerchant(merchantId: string) {
  // 1. 创建默认服务
  const defaultServices = [
    { name: '洗剪吹', duration: 60, price: 50 },
    { name: '染发', duration: 120, price: 200 },
    { name: '烫发', duration: 180, price: 300 },
  ]
  await ServiceModel.insertMany(
    defaultServices.map(s => ({ ...s, merchant_id: merchantId }))
  )
  
  // 2. 创建默认员工（发型师）
  await StaffModel.create({
    merchant_id: merchantId,
    name: '发型师1',
    title: '首席发型师',
    is_active: true
  })
  
  // 3. 初始化计数器
  await CounterModel.create({
    merchant_id: merchantId,
    date: new Date().toISOString().slice(0, 10),
    count: 0
  })
}
```

## 10. 技术要点

### 10.1 数据隔离

所有门店级别的查询都需要加上 `merchant_id` 过滤：

```typescript
// 中间件自动注入 merchant_id
app.use(async (ctx, next) => {
  if (ctx.state.user?.role === 'merchant_admin') {
    ctx.state.merchantId = ctx.state.user.merchant_id
  }
  await next()
})

// 查询时自动过滤
const appointments = await AppointmentModel.find({
  merchant_id: ctx.state.merchantId,
  // ... 其他条件
})
```

### 10.2 店长账号创建逻辑

```typescript
// 审核通过时创建店长账号
async function createMerchantAdmin(merchantId: string, phone: string, name: string) {
  const username = `owner_${merchantId.slice(-6)}`
  const tempPassword = generateTempPassword() // 随机8位密码
  
  const admin = await AdminModel.create({
    username,
    password_hash: await hashPassword(tempPassword),
    real_name: name,
    phone,
    role: 'merchant_admin',
    merchant_id: merchantId,
    type: 'merchant',
    is_active: true
  })
  
  // 发送短信通知
  await sendSMS(phone, {
    template: 'MERCHANT_ADMIN_CREATED',
    params: { username, tempPassword }
  })
  
  return { username, tempPassword }
}
```
